# Location Insights App

A beginner-friendly web application that allows users to input geographic
locations (latitude and longitude), visualize them on an interactive map,
and generate simple insights using charts.

This project is designed to demonstrate core front-end development,
data validation, and basic analytics concepts.

---

## Features

- Input latitude and longitude with validation
- Accepts only numeric coordinate values
- Plots locations on an interactive map
- Logs added locations with timestamps
- Displays location data in a table
- Generates basic charts for insights
- Allows exporting data and visuals

---

## Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)
- Leaflet.js (maps)
- Chart.js (charts)
- html2canvas (export visuals)

## How to Run the App

1. Clone the repository
2. Open `index.html` in a browser
3. Enter latitude, longitude, and place name
4. View locations on the map and in the table
